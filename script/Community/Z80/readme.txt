Z80-microprocessor-personal-script
Script per il microprocessore Z80 con linguaggio proprietario

Informazioni:
Z80++ è sviluppato in c++ e utilizza le API di AXIC SCHOOL BY IT, per poterlo usare è necessario disporre di una mainboard con installato AXIC OS + AXIC Plugin.

All'interno del codice sono presenti riferimenti al codice di AXIC, non è consigliabile dunque avviare lo script in un microprocessore non installato su una mainboard AXIC.

Tutta via è possibile modificare lo script e addatarlo alle proprie necessità.


AXIC è una piattaforma closed sviluppata da BY IT che permette la virtualizzazione dei BUS presenti nelle memorie del mcp z80 utilizzando semplicemente un computer.

Il mio script utilizza queste funzioni per la programmazione di un'intero sistema di mcp(microprocessori) senza l'ausilio di una mainboard con programmatore integrato.

Tuttavia richiede una mainboard con installato AXIC.
